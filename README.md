# Sales Skills

Claude Code skills for sales, marketing, and GTM workflows.

## Quick Start

```bash
npx skills add sales-skills/sales
```

## What is `sales-do`?

`/sales-do` is a router skill. Describe your sales objective in plain language and it matches you to the right specialized skill with a ready-to-use prompt. Think of it as the entry point ŌĆö you don't need to memorize which skill does what.

```
/sales-do write a cold outbound sequence for CFOs at mid-market fintech companies
```
