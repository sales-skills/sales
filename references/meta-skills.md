# Meta / Coordinating Skills Registry

Cross-platform strategy skills that coordinate execution across multiple tools for a specific problem domain. When adding a new platform, check which meta skills it belongs to and update them.

A platform can belong to **multiple** meta skills (e.g., a platform with email + SMS + chat belongs to `sales-email-marketing`, `sales-sms-marketing`, and `sales-chatbot`).

## How to use this registry

When adding a new platform via `/add-platform`:

1. **Match the platform's capabilities to meta skills** — a platform with email campaigns belongs in `sales-email-marketing`, one with SMS in `sales-sms-marketing`, etc. A platform can match multiple meta skills.
2. **Update each matched meta skill** — add the platform to comparison tables, "In {Platform}" sections or `references/platforms.md`, and related skills.
3. **If no meta skill exists for a capability** — propose creating one. For example, if a platform offers "webinar" and no `sales-webinar` existed, create it.
4. **Update this registry** — add the platform name to each meta skill it was added to, and add any new meta skills created.
