# HyNote AI Platform Reference

## Platform overview

HyNote AI (by Turing Intelligence LLC) is a full-stack AI note-taking platform that captures audio, meetings, files, YouTube videos, and web pages, then transforms them into structured notes, flashcards, quizzes, mind maps, flowcharts, presentations, and AI-narrated podcasts. Cross-platform (web, iOS, Android, tablet, Apple Watch). GDPR, CCPA, and HIPAA compliant. Targets five segments: students/academics, technical/advisory professionals, business professionals, content/media professionals, and accessibility users.

## Key modules

### Audio recording & transcription
- Record meetings, lectures, phone calls, and ambient audio
- Real-time transcription (Plus+ plans only)
- Speaker identification (Plus+ plans only)
- Apple Watch recording capability
- Automatic Zoom meeting transcription via calendar integration

### File processing
- Upload text, PDF, images (OCR), videos
- YouTube video transcription via URL paste
- Web page summarization via URL
- Chrome Extension for web capture

### AI note generation
- Automatic summarization and key point extraction
- Action item identification
- Professional templates: FAQ, resume, outline, case study, SWOT analysis
- Cross-device synchronization (web, mobile, tablet)

### Study material generation
- Flashcard generation from notes
- Quiz generation from notes
- Mind maps and flowcharts
- AI-narrated podcast generation (uses credits)
- Presentation/slide generation (uses credits)
- Infographic generation (uses credits)

### Content export
- PDF export (all plans)
- Transcript export (Pro+ only)
- Notes transfer to Google Docs, Notion (Plus+ only)
- Blog/podcast exports

## Comparison with alternatives

### vs Otter.ai
- **HyNote wins**: Multilingual accuracy (more frequent algorithm updates for accents/language transitions), bot-free recording option, file processing (PDF/image/video/YouTube), study material generation (flashcards, quizzes)
- **Otter wins**: Deeper third-party integrations, real-time collaboration features, stronger Zoom integration, real-time captions, public API (Enterprise)

### vs Granola
- **HyNote wins**: Cross-platform (Granola is Mac-only), file processing, study materials, multi-format output
- **Granola wins**: Simpler UX for light users, bot-free by design, AI-enhanced notes (you type, AI augments), CRM integration (HubSpot/Attio/Affinity on Business+)

### vs Coconote
- **HyNote wins**: Meeting recording (calendar integration, Zoom/Meet/Teams), web page summarization, YouTube URL paste, PDF/image processing, mind maps, presentations, Apple Watch recording, more integrations (Notion/Google Docs)
- **Coconote wins**: Backed by Quizlet (established brand), purpose-built for students, folder-scoped AI chat across all notes, Apple Vision Pro support

### vs Voicenotes
- **HyNote wins**: Meeting recording (calendar integration), file processing (PDF/image/video), study materials (flashcards, quizzes), mind maps and presentations
- **Voicenotes wins**: Obsidian sync, Zapier automation (9 triggers), Ask AI semantic search across all notes, lower price ($14.99/mo vs $15.83/mo for comparable features)

## Known issues

- One Product Hunt reviewer reported misleading refund support — directed to App Store refund when purchase was made on website
- Feature requests for more templates and cloud storage for long recordings
- Advanced features (speaker ID, real-time transcription, Notion export) gated behind Plus+ plans
- Credits are one-time grants, not monthly refills — can run out unexpectedly
