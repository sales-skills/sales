# Noted Platform Reference

## Platform overview

Noted (by Digital Workroom Ltd, UK) is a combined audio recording and rich-text note-taking app for Apple devices. Its key differentiator is **timestamped linking** — every line you type or draw during a recording is tied to that moment in the audio, letting you tap any note to jump to that exact point in playback. With 3M+ downloads and an App Store Editors' Choice badge, it targets students, professionals, and journalists who need simultaneous audio capture and written notes. Available on iPhone, iPad, Mac, Apple Watch, and Apple Vision Pro (plus a secondary Android version).

## Key modules

### Audio recording
- Records audio while you take notes simultaneously
- Crystal-clear recording with background noise reduction
- Multiple recording quality options
- Import external audio/video files into notes
- Audio trimming for removing unwanted sections

### Time-stamped notes
- Every line typed during recording gets an automatic timestamp
- Tap any timestamp during playback to jump to that moment in the audio
- Rich-text editing with formatting (bold, italics, lists, etc.)
- Adjustable margins and highlighting

### AI Transcribe (Noted+ only)
- Converts recordings into text using AI
- Supports 18 languages
- Cloud-based processing (requires upload)
- Transcription UI is in the note's options menu — not prominently placed (common user complaint)

### Intelligent Playback
- Machine learning detects and skips background noise and silence
- Jumps straight to meaningful parts of recordings
- Useful for reviewing long lectures or meetings

### Apple Pencil & Drawing (iPad)
- Draw and doodle directly in notes during recording
- Drawings also receive timestamps
- Time Stroke: replay drawings frame-by-frame in sync with audio

### Voice typing / Dictation
- Transform spoken words into typed text in real-time
- Separate from AI Transcribe — dictation is live, transcription is post-recording

### Ask AI (Noted+ only)
- Search recordings for answers instantly
- AI summaries converting lengthy recordings into key points

### Apple Watch recording
- Start and control recordings from Apple Watch
- Hands-free recording for in-person meetings

## Integrations

**Native integrations:** None. Noted is a standalone Apple ecosystem app.

**Export options:**
- PDF — formatted note with timestamps
- TXT — plain text export
- M4A — raw audio file
- Web links — shareable note links (no recipient sign-up required)
- iCloud sync — automatic across all Apple devices

**No CRM connectors, no Zapier, no webhooks, no API.** For automated workflows, export audio and use external tools.

## Workflow setup

### Typical lecture recording workflow
1. Open Noted → create a new note
2. Tap the red record button to start recording
3. Type or draw notes as the lecture progresses — each line gets a timestamp
4. After class, use Intelligent Playback to review (skips silence)
5. Use AI Transcribe (Noted+) to generate a full transcript
6. Export to PDF for studying or share via web link

### Meeting note workflow
1. Create a note before the meeting
2. Start recording when the meeting begins
3. Type key points, action items, or questions during the meeting
4. After the meeting, tap any timestamp to replay that specific moment
5. Use Ask AI to search for specific topics discussed
6. Export summary as PDF or share via web link

### Apple Watch quick capture
1. Open Noted on Apple Watch
2. Start recording (hands-free)
3. Later, open the synced recording on iPhone/iPad/Mac
4. Add written notes and timestamps
5. Use AI Transcribe to generate text
