# Simular (Sai) Learnings

Accumulated tips, gotchas, and discoveries from real usage. Append new findings with today's date.

## 2026-04-26 — Initial research

- Sai pricing changed significantly: Starter was $200/mo, now $20/mo. The pricing page may still show legacy pricing in some caches.
- The "full API access" on Pro plan is not well-documented. The primary developer surface is Simulang scripting and the Browser Python API, not a REST API.
- Agent-S requires `tesseract` OCR installed via Homebrew as a dependency.
- Google Sheets is the only app with native Simulang actions (cell read/write). Everything else is GUI-based.
- Product Hunt reviews mention difficulties with Chinese text input and Google CAPTCHA blocking.
